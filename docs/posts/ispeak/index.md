# 简单介绍

## 介绍

ispeak 模块是一个用于发布说说/想法的模块。

- 可设置可见权限
- 标签与speak进行关联
- 可对speak配置评论系统
- 个性化标签进行分类
- 多端发布示例（理论上只要能请求API就可以发布）
- ...

## 发送及管理端

目前解锁发送姿势

1. IOS端

   - 快捷指令
   - Taio App脚本

2. Windows/Linux/Mac

   - [网页端](https://github.com/kkfive/speak-biubiu)
   - [uTools脚本](https://github.com/kkfive/speak-biubiu/blob/master/scripts/utools.js)

3. Android

   可使用网页端或其他方案

目前管理端只有kkadmin进行管理
